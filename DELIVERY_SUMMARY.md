## 🚀 Quick Start (5 Minutes)

```bash
# 1. Create project
npm create vite@latest gqf-website -- --template react
cd gqf-website

# 2. Install dependencies
npm install react-router-dom framer-motion lucide-react axios
npm install -D tailwindcss postcss autoprefixer @tailwindcss/forms @tailwindcss/typography

# 3. Setup Tailwind
npx tailwindcss init -p

# 4. Copy provided files to your project
# - All .jsx files to src/components/ and src/pages/
# - globals.css to src/styles/
# - tailwind.config.js to project root
# - Update App.jsx in src/

# 5. Update package.json with provided version

# 6. Run
npm run dev
```

**Now you have a running website!** 🎉

## 🎯 How to Customize

### 1. **Colors**
Update `tailwind.config.js`:
```javascript
colors: {
  'dark-bg': '#0f1419',  // Change any color
  'accent-gold': '#C5A059',
  // ... etc
}
```

### 2. **Typography**
Edit `globals.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=YourFont...');
```

### 3. **Content**
Update text in each component:
- Headlines, descriptions
- Replace images in `public/assets/`
- Update contact information
- Modify form fields

### 4. **Features**
Add new sections to `Home.jsx`:
```jsx
<motion.section>
  <YourNewComponent />
</motion.section>
```

## 📞 Troubleshooting

### "Components not rendering"
→ Check import paths and folder structure

### "Styles not applying"
→ Verify Tailwind classes are spelled correctly
→ Check tailwind.config.js is in root directory

### "Dashboard not showing"
→ Update the iframe src URL with your dashboard ID
→ Ensure dashboard is publicly shared

### "Form not submitting"
→ Check backend endpoint is running
→ Verify environment variables
→ Check browser console for errors

### "Mobile layout broken"
→ Check responsive classes (sm:, md:, lg:)
→ Test with actual mobile device
→ Check viewport meta tag is in HTML
